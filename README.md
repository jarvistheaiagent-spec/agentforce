# Agentforce / Data Cloud Project

This repository bootstraps a Salesforce DX (SFDX) project intended for building:
- Agentforce agents + actions
- Data Cloud (Data360) configuration + orchestration
- Supporting Apex/LWC/Flows/integrations

## Getting started

1. Install Salesforce CLI (`sf`).
2. Authenticate to the target org.
3. Deploy metadata.

> This repo is currently a scaffold; implementation will follow the specifications.
